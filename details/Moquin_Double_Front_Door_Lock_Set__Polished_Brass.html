Moquin Double Front Door Lock Set

Elevate the style of your entryway with the Moquin Double Front Door Lock Set, crafted from premium zinc alloy and finished in polished brass for a timeless, elegant look. This robust hardware set is built to last, offering durability that stands up to daily use and ensures long-term performance.

Featuring a combination of a deadbolt and a thumb-turn lock, this set provides exceptional security without compromising ease of entry. The adjustable through-bolt allows for flexibility with door thicknesses ranging from 1-3/8'' to 1-3/4'', and can also be vertically adjusted by up to 1.2'' to fit your existing door hardware perfectly.

Whether you're securing your home or enhancing the appearance of a commercial space, this lock set delivers both style and strength. Its reversible handle design makes installation effortless, and the one-tool setup ensures a quick, hassle-free experience for anyone, from DIY enthusiasts to professional installers.

Key Features

  • Heavy-duty zinc alloy construction for long-lasting durability
  • Adjustable through-bolt for 1-3/8'' to 1-3/4'' door thicknesses
  • Polished brass finish for a refined and elegant appearance
  • Includes deadbolt with key and thumb-turn for enhanced security
  • Easy one-tool installation for quick and hassle-free setup
